# Calendar Work Shifts Tool

## Description 
This tool allows the user to create a calendar event for work shifts. 

The user may select which pre-exsiting calendar the events can be added to, and the event creation is done one week at a time from Monday to Saturday. The possible shifts are:
- Morning (8am-4pm)
- Mid (10am-6pm)
- Afternoon (12pm-8pm)
- N/A (off)
